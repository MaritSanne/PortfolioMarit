MAAS verbeteringsplan

Voor het vak User Centred Design kreeg ik de opdracht om een verbeteringsplan te maken voor de MAAS koffiemachines die zich bevinden op de Hogeschool van Amsterdam. 

Klantnaam MAAS, Hogeschool van Amsterdam
Duur 3 weken
Datum 14-11-2022 t/m 10-12-2023

Beschrijving

Voor dit project heb ik voor "MAAS" een UX ontwerp verbeteringsplan gemaakt. De MAAS machines werken niet optimaal en deze heb ik dus onderzocht. Dit verbeteringsplan wordt dus gebaseerd op het onderzoek wat ik heb gedaan. Hierdoor wordt de User Experience verbeterd met de koffiemachines.

Allereerst begon ik met het maken van een moodboard en storyboards. Een storyboard is een visuele weergave van een gebeurtenis. Ook heb ik in de eerste week een user trip gedaan, om mij in te leven in de gebruiker. Ik heb zelf een kopje koffie gehaald uit de machine om de frustraties zelf te ontdekken. Vervolgens heb ik ook nog een Deep Hanging Out gedaan. Bij een Deep Hanging Out ga je naar de locatie van de HvA bij een MAAS machine gebruikers observeren.

In de tweede week heb ik een kennisclip presentatie gegeven. In deze kennisclip kwamen begrippen terug die handig zijn voor het onderzoek. We werden in groepjes verdeeld en ieder groepje moest een andere week een kennisclip voorbereiden. Bij deze kennisclip moesten we ook per week 4 vragen met het antwoord aan onze productbiografie toevoegen.

In de laatste week van het vak UCD heb nog meer verschillende onderzoeken gedaan, namelijk het maken van een affinity map, een empathy map, een persona en een experience map. Ook heb ik in de laatste week al mijn feedback toegepast en de samenvattende poster gemaakt voor de eindpresentatie.

Resultaat / Conclusie

Voor de eindoplevering moest ik een productbiografie inleveren met al mijn onderzoeken. Ook heb ik een korte presentatie moeten geven. Hierbij moest ik een samenvattende poster maken van de gehele productbiografie. Mijn uiteindelijke cijfer voor het vak User Centred Design is een 7,0.

Ik ben erg tevreden met dit cijfer. Ik ben erg trots op het eindresultaat van dit project, omdat ik nog weinig ervaring had met UX en ik met dit vak veel van mijn competenties heb kunnen toepassen, zoals teamwork, probleemoplossend vermogen en flexibiliteit.
